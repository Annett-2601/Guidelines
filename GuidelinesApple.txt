App Store Review Guidelines

1.	Безопасность 

Приложения не должны: 
- содержать оскорбительный контент касающийся дискриминация, религии.
- содержать Изображения насильственного характера
- содержать Изображения, поощряющийся незаконное использование оружия
- содержать Материалы порнографического характера
- разрешают анонимные телефонные звонки, розыгрыши, или обмен сообщениями
-недолжны передавать личную информацию и информацию об устройстве третьим лицам


Приложения с пользовательским контентом должны содержать
- фильтрацию материалов, которые пользователь не хочет просматривать
- возможность сообщения об оскорбительных материалах и своевременных ответах на вопросы
- опубликована контактная информация, чтобы пользователь мог легко связаться с вами.

Детская категория
  - приложения в этой категории не должны содержать стороннюю аналитику и рекламу.
 
Физический вред
-Медицинские приложения должны получить разрешение регулирующих органов
- Медицинские приложения должны напоминать пользователям о необходимости консультации с врачом
- Приложение поощряющие употребление табака, наркотиков, электронных сигарет, употребления алкоголя запрещены.


2.	           Производительность

-Материалы для проверки приложений должны быть с окончательными версиями, со всеми необходимыми метаданными и работающими URL адресами)
- Демоверсии, бета-версии пробные версии не размещаются в магазине, для них предназначен TastFlight, причем тестироваться они должны бесплатно
	-  Точные метаданные -описание приложения, скриншоты, информацию о конфиденциальности.
	- Если приложение имеет дополнительные платные функции, необходимо убедится, что в описании, снимках экрана и предпросмотрах четко указано, что требуется дополнительные покупки для каких -то элементов.
	- снимки экрана должны показывать не только обложку приложения, но и используемое приложение.
	- Размещать свое приложение в наиболее подходящую категорию. Приложение можно разместить в основную и дополнительную категорию, исключения составляют приложения попадающие в категорию дети. Они могу располагаться дополнительно еще в 2 категориях.
	-Если приложение является игрой, то оно может располагаться в двух подкатегориях.
	- Необходимо честно-выставлять возрастной рейтинг, чтоб приложение правильно согласовалось с родительским контролем.
	- На скриншотах должна отображаться вымышленная информация об учетных данных человека
	-Приложения должны четко описывать новые функции в тексте «Что нового»
-приложения, которые меняют настройки громкости или отключают функции стандартных переключателей будут отклонены.
- Запрещается оправлять приложения с пустыми рекламными баннерами или текстовой рекламой.
-Приложения должны запрашивать согласие на использование камеры, микрофона, записи экрана.


3.	Бизнес
- Запрещается повышать рейтинг поддельными отзывами
-Могут отказать, если посчитают, что цена на ваше приложение завышена
- Сроки действия любых внутри игровых валют, приобретенных через приложение не могут истекать
-Если у приложения есть пробная версия, то длительность пробной версии должна быть четко указана.
- Приложения не должны заставлять пользователя оценивать приложение.

4.	Дизайн 
-Приложение должно работать само по себе без дополнительной установки другого приложения
-Приложения, должны предлагать вход с помощью учетной записи APPLE, если они используют стороннюю службу для  входа- соц сети

5.	Юридические
-Приложение должно соответствовать всем местным законам
-Приложения должны содержать ссылку на политику конфиденциальности.
-Приложения, которые собирают данные о пользователях должны обеспечивать согласие на сбор данных
-нельзя использовать приложения для тайного получения паролей или других личных данных.
-данные собранные из приложений могут быть переданы третьим лицам только для улучшения приложения или показа рекламы с соответствии с лицензионным соглашением.
-приложения для детей не должны включать стороннюю рекламу.
- приложение должно только созданный вами контент, или на использование есть лицензия.
