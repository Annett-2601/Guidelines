Навигация
	- Приложение поддерживает навигацию по кнопке назад
	- Приложение поддерживает навигацию с помощью жестов для возврата /перехода на главный экран
	
Уведомления:
 	- Не прерывать работу пользователя посреди задачи уведомлениями о просьбе о просьбе оценить нашу работу.
	- Не отправлять поздравления с праздниками в виде уведомлений
	- В уведомлении при загрузке обновления в целях экономии батарее должна быть кнопка отмены.
Как выглядят уведомления.
•	Значок приложения.
•	Название приложения
•	Текст заголовка (необязательно)
•	Время (не обязательно)
•	Индикатор, появляется, если сообщение можно развернуть.
•	Если уведомление пришло из соц. Сети, то  аватар
•	Кнопки для выбора действий

Уведомления приходят в ящик уведомления, первыми размещаются более ранние уведомления
•	Свайпом можно убрать уведомления, если разрешено
•	Разрешен ввод текста в уведомление, если тект большой, то можно его перенаправить в приложение.

Уведомления бывают
 	транзакционные 
•	Взаимодействие человека с человеком (входящие звонки, сообщения)
•	Уведомления, улучшающие повседневную жизнь (напоминания, будильники) 
•	контроль за переходными состояниями устройства (воспроизведение музыки, работа секундомера)

 не транзакционные (необязательные)
UI  и графика
•	поддержка альбомной портретной ориентации ( если возможно)
•	приложения используют весь экран в обеих ориентациях
•	приложение правильно срабытывает при переходах между ориентациями.
